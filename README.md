# My Learning Project

A Python project for learning systems building, Pandas, AI, and data management.

## Setup

```bash
# Create virtual environment
python -m venv .venv

# Activate it
# Windows:
.venv\Scripts\activate
# Mac/Linux:
source .venv/bin/activate

# Install dependencies
pip install -e ".[dev]"
```

## Usage

```bash
python -m my_project
```

## Testing

```bash
pytest
```
