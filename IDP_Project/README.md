# IDP - Electrical Component Detection

OpenCV-based project for detecting and classifying electrical components.

## Project Structure

```
IDP_Project/
├── data/
│   ├── raw/         # Raw images for training/testing
│   └── processed/   # Preprocessed images
├── models/          # Trained models
├── notebooks/       # Jupyter notebooks for experiments
├── src/             # Source code
└── tests/           # Unit tests
```

## Setup

```bash
pip install opencv-python numpy matplotlib scikit-learn
```

## Usage

```bash
python src/main.py
```
